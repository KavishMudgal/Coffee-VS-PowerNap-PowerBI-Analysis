# Coffee-VS-PowerNap-PowerBI-Analysis
This Power BI dashboard analyzes the impact of Coffee and Power Naps on alertness levels and productivity performance.

## Project Overview

This project presents a comparative analysis of Coffee consumption and Power Naps to evaluate their effectiveness in improving alertness and productivity.

Using Power BI, the dashboard analyzes alertness spikes, sustained focus duration, and productivity trends across different sleep durations and occupations.

## Dashboard Features

- Alertness Before vs After Comparison
- Duration-Based Alertness Trends
- Sleep Hours vs Productivity Analysis
- Occupation-wise Performance Insights

## Key Findings

- Power Naps generate higher short-term alertness peaks.
- Coffee provides a more stable and slightly longer sustained alertness effect.
- Productivity outcomes are significantly influenced by sleep duration.
